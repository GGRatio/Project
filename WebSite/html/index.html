Creation

В ходе выполнения лабораторной работы 12 наша команда создала приложение помощник на языке программирования С++ в консоли. Приложение является универсальным инструментом. Каждый участник сыграл важную роль в создании это проекта. Мы использовали Agile/Scrum, что подразумевает наличие спринтов.

Manual

  • Для выбора модуля необходиммо ввести соответсвующий номер
  • В случае возникновения проблем просто закрыть консоль, а затем запустить приложение повторно
  • Для прекращения работы ввести 4
